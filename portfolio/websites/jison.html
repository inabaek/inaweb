Ceramist Jison's Website Project

As one of the talented professional ceramists in Korea, I knew something to be done after noticing that Jison needs her website for presenting her pieces and potentially make a digital market. The site was built through website builder Wix. This allowed Jison to make changes to the content of the posts and add following events through an easy-to-use interface. This project is ongoing and I am very excited to see this website is being used by Jison soon!
